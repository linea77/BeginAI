02 머신러닝의 학습 방법 살펴보기

1. 지도학습(Supervised learning)

- 정답(레이블, label)이 있는 데이터를 이용하여 학습하는 것
- 데이터 vs 레이블(Label)
→ 데이터 : 사과 사진 50개
→ 레이블 : '사과', '배'라는 이름 = 정답

사례: 어디에 사용되는가?

1) 분류(classification)

- 이진분류(binary classification)
- 정답(레이블)이 2개인 경우
- 예1) 스팸메일인지, 일반 메일인지 구별
- 예2) 환자의 폐 CT사진에서 암이 보이는지, 아닌지 구별

- 다중분류(multiclass classification)
- 정답(레이블)이 3개 이상인 경우
- 예1) 새의 종류 구별
- 예2) 붓꽃(Iris)의 종류 구별
- 예3) 지폐의 종류 구별
- 예4) 사람의 나이대 구분(10대/20대/30대/...)

2) 회귀(regression)

-회귀란?: 연속적인 값을 예측하는 것
- 예1) 집값 예측
- 예2) 학생의 성적 에측
- 예3) 주식 예측
- 예4) 사람의 나이 판단(23세/24세/..45세


2. 비지도학습(Unsupervised learning)

- 정답이 없는 데이터를 사용하여 인공지능이 스스로 특징을 학습하는 방법
- 지도학습이 불가능한 상황(모범 답안이 없는 경우) 매우 유용하게 사용됨
- 현실 세계에는 정답이 있는 데이터보다 정답이 없는 데이터가 많음.

사례: 어디에 사용되는가?

1) 군집화(Clustering)

- 데이터를 여러 그룹으로 묶는 것
- 예1) 콘텐츠 혹은 상품 추천 알고리즘
1. 사람들의 구매 내역을 바탕으로 몇 개의 그룹(예 : A, B, C)으로 나눔
2. 새로운 이용자의 구매 데이터를 통해 어떤 그룹(예: A)에 속하는지 확인
3. 그 그룹(예: A)의 다른 사람이 이전에 어떤 물건을 구매하였는지 확인하고, 이용자에게 이를 추천함

2) 차원 축소(dimensionality reduction)

- "차원" : 데이터의 특징(feature)을 의미함
- 즉, 데이터의 여러 특징들 중에서 가장 대표적인 특징만 뽑아내는 것.

3. 강화학습
